 Notes REST API

A simple REST API built with Node.js and Express.

## Features
- Create a note
- Get all notes
- Get a single note
- Update a note
- Delete a note

## Tech Stack
- Node.js
- Express

## Endpoints

GET /notes  
POST /notes  
GET /notes/:id  
PUT /notes/:id  
DELETE /notes/:id
